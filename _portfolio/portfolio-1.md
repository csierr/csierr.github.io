This project demonstrates the fine-tuning of the BETO model (a BERT-based model for Spanish) to classify sentences into school subjects.  

It includes training, evaluation, and full integration into a local Flask application with a basic HTML/JS interface.

Ideal for those starting with transformers in Spanish NLP tasks.
