# csierr.github.io

Hi! 👋 This repository contains the source code for my personal academic website (**work in progress!** continuously making
improvements), built with the [AcademicPages](https://github.com/academicpages/academicpages.github.io) template — a flexible Jekyll-based theme designed for researchers, students, and professionals.

### 🌟 About this site

The goal of this site is to share a bit about who I am, what I’m working on, and the topics that inspire me. You’ll find:

- A brief overview of my background and current work.
- Experiences and projects in areas like AI, NLP, LLMs, and IoT.
- Ongoing research.
- Links to my CV, contact info, and other platforms.

I created this site to have a simple space to reflect my journey, keep learning, and hopefully connect with people working on meaningful tech 🧠💡

📍 Live site: [csierr.github.io](https://csierr.github.io)
